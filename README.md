##Gulp-create Folders

Description:
    For creation of files default for especific module, in estructure:
    use --n to inform module name in app
    
    <code>
    www/
    └── app
        └── products
            ├── products-controller.js
            ├── products-service.js
            ├── products-view.html
            └── products.css
    </code>
<h1>Usage</h1>

gulp create-folders --n NameOfModule
